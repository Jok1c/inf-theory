# Система шифрования файлов

Реализация двух алгоритмов шифрования для русского языка:
- Метод децимаций
- Алгоритм Виженера с самогенерирующимся ключом

## Запуск

```bash
npm install
npm run build
npm run serve
```

Приложение откроется в браузере на `http://localhost:8080`

## Использование

1. Выберите метод шифрования
2. Введите ключ:
   - Для децимаций: число от 1 до 32 (взаимно простое с 33)
   - Для Виженера: текст на русском языке
3. Выберите файл для обработки
4. Нажмите "Зашифровать файл" или "Расшифровать файл"
5. Результат автоматически скачается
