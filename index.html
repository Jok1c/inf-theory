🔐 Система шифрования файлов

Метод децимаций и алгоритм Виженера с самогенерирующимся ключом

Для метода децимаций: число от 1 до 32 (взаимно простое с 33)
Для алгоритма Виженера: текст на русском языке
Поддерживаются текстовые файлы (.txt)

📄 Файл будет обработан, и результат будет автоматически скачан.

⚠️ Обрабатываются только русские буквы, остальные символы игнорируются.
